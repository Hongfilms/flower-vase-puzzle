# flower-vase-puzzle

## README.md

```markdown
# 꽃병 퍼즐 게임

이 프로젝트는 아름다운 꽃병 이미지를 사용한 간단한 9조각 슬라이딩 퍼즐 게임을 구현한 웹 애플리케이션입니다.

## 기능

- 9개의 조각으로 나뉜 꽃병 이미지 퍼즐
- 퍼즐 조각을 클릭하여 이동
- '섞기' 버튼으로 퍼즐 재배치
- 퍼즐 완성 시 알림 표시

## 사용 방법

1. 이 리포지토리를 클론하거나 다운로드합니다.
2. `index.html` 파일을 웹 브라우저에서 엽니다.
3. 퍼즐 조각을 클릭하여 이동시키고, 원래 이미지를 완성하세요.
4. '섞기' 버튼을 클릭하여 언제든 퍼즐을 다시 섞을 수 있습니다.

## 기술 스택

- HTML
- CSS
- JavaScript (바닐라)
```
